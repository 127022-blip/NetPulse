## 🚀 编译和运行步骤

### 第一步：安装 XcodeGen

XcodeGen 是用于生成 Xcode 项目的工具。

**方法一：使用 Homebrew（推荐）**
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install xcodegen
```

**方法二：手动下载**
```bash
# 从 https://github.com/yonaskolb/XcodeGen/releases 下载最新版本
# 解压后将 xcodegen 可执行文件放到 /usr/local/bin/ 目录
```

### 第二步：生成 Xcode 项目

```bash
cd /Users/genglei/我的文件/NetPulse
xcodegen generate
```

成功后会看到 `NetPulse.xcodeproj` 文件。

### 第三步：编译项目

**使用 Xcode 打开：**
```bash
open NetPulse.xcodeproj
```
然后在 Xcode 中点击 ▶️ 运行按钮。

**或使用命令行编译：**
```bash
xcodebuild -project NetPulse.xcodeproj -scheme NetPulse -configuration Debug build
```

编译成功后的 .app 文件位于：
```
~/Library/Developer/Xcode/DerivedData/NetPulse-xxx/Build/Products/Debug/NetPulse.app
```

## 🎯 功能特性

### ✅ 已实现功能

1. **菜单栏常驻显示**
   - 实时显示网络状态图标
   - 支持多种显示模式（仅图标、图标+速度、图标+上下速度）

2. **下拉面板**
   - 实时网速监控（下载/上传）
   - 今日流量统计
   - 网络状态指示
   - 活跃接口显示

3. **设置面板**
   - 菜单栏显示模式选择
   - 更新频率调整
   - 通知开关
   - 速度告警阈值
   - 网络接口选择

4. **通知功能**
   - 断网告警
   - 网络恢复通知
   - 速度低于阈值告警

5. **数据持久化**
   - 今日流量统计（自动每日重置）
   - 历史流量记录（保留30天）

### 🔧 技术实现

| 技术点 | 实现方式 |
|--------|----------|
| 菜单栏 | NSStatusItem + NSPopover |
| UI 框架 | SwiftUI |
| 架构模式 | MVVM |
| 网络监控 | NWPathMonitor + netstat 命令 |
| 数据存储 | UserDefaults + SQLite |
| 通知 | UserNotifications |

## ⚠️ 注意事项

1. **权限要求**
   - 应用需要网络权限（已在 entitlements 中配置）
   - 通知功能需要用户授权

2. **系统要求**
   - macOS 12.0 (Monterey) 或更高版本

3. **菜单栏应用**
   - 这是一个纯菜单栏应用，运行后会在菜单栏显示，不会出现 Dock 图标
   - 应用会在后台持续运行

## 🐛 常见问题

### Q: xcodegen generate 报错？
**A:** 确保已正确安装 XcodeGen，并检查 project.yml 语法是否正确。

### Q: 编译报错找不到文件？
**A:** 确保所有目录和文件都在正确位置，特别是 Assets.xcassets 内的 Contents.json。

### Q: 应用启动后菜单栏没有显示？
**A:** 检查是否成功编译，尝试重新编译并运行。
